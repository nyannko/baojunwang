Regulated Promoters


A set of promoters to be used as circuit inputs or processing.

Input promoters have regulators with inducible activity by small molecules (synthetic or quorum-sensing related), light or metal ions.

Processing promoters, used for NOT (repressors) or Buffer (activators) gates, don't have appointed inducers.

Please also note that values for absolute expression are not standardized under the same units, so should not be compared.
