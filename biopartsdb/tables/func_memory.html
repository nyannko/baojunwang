Memory


Parts for systems where a temporary input can set a constant memory output.

Recombinases represent the most proeminent candidates for such a behaviour, because inversion of DNA molecules will be retained without the recombinase's presence. The system can be reset with RDF proteins, but most recombinases don't have cognate RDFs.
