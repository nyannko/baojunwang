Fernandez-Rodriguez, J. & Voigt, C. A. Post-translational control of genetic circuits using Potyvirus proteases. Nucleic Acids Res. 44, 6493â€“6502 (2016).

In this publication, Fernandez-Rodriguez and colleagues utilize protease tags to expose an N-terminal degradation tag (N-end rule based) and thus enhance degradation of the attatched protein. The reported efficiencies are based on a measurement made with tagged GFP, but the publication also characterizes the same system better with tagged repressors (TetR and PhlF). That activity was not reported here because it depends on more vaariables rather than isolating the tag's influence.

There is a more recent work characterizing a larger set of protease tags, but on mammalian cells. They should work on E.coli, but weren't chosen for the repository because characterization on E. coli is preferred.
